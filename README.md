# Smart Back Office Automation System

A Core Java console-based application to automate back-office workflows using rule-based AI logic.

## Project Structure
- src/ → Java source files
- database/ → SQL table schema

## Entry Point
- MainApp.java contains the main() method

## How to Run
1. Execute database/smartops_db.sql in MySQL
2. Compile all Java files in src/
3. Run MainApp.java

## Features
- Employee leave request submission
- Automatic approval based on rules
- Admin dashboard to view requests
